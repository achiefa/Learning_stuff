template<class T , class Dependent = std::nullptr_t>
T * injector::get_instance ( ) const
inline

Function template that returns a pointer to the instance of an object in the injector.

When calling injector.get_instance<T>() the function will return a pointer to the instance of the object T in the injector with the dependencies already resolved. It also means that the object T and the dependencies must be registered with the injector. Failing in that results in an exception.

template<class InstanceType , class Deleter , class ... Deps>
std::unique_ptr< InstanceType, Deleter > injector::inject ( instance_factory_function< InstanceType, Deleter, Deps ... > instance_factory) const
inline

The inject function invokes the factory function pointed by instance_factory. Each parameter of the factory function is bound to the appropriate service (object) instance. It will return the std::unique_ptr specified by the factory.

The ellipsis at the end of the call are needed in the context of variadic template. They cause the expression to be repeated for each element in the parameter pack Deps. For example, if the instance factory passed to inject looks like

code
std::unique_ptr<Service1> Service1Factory(Service2* s2, Service3* s3) {
return std::make_unique<Service1>(s2, s3);
}
